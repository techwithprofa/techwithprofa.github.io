ExamLLB: Interactive Learning Game

A Gamified Exam Preparation Platform for Law Students

Project Completed

Project Overview

ExamLLB is a web-based, gamified learning platform built with Next.js to help law students prepare for exams. It features a roadmap-driven multiple-choice game with timed questions, scoring, hints, and instant feedback. The core innovation is the integration of an AI to generate personalized learning roadmaps, guiding users through exam content efficiently and improving engagement.

The project was conceptualized to help users with only 4-5 weeks to prepare, focusing on a "brain-friendly" system that makes learning effective and intuitive.

Development Journey

Phase 1: Research & AI-Powered Roadmap Completed

  • Researched exam content and gathered necessary data
  • Designed a learning roadmap concept to guide users
  • Utilized an AI (Claude) to generate JSON-based roadmaps from data

Phase 2: Game Development & Deployment Completed

  • Built the MCQ game with timed questions and a scoring system
  • Developed the full web application using Next.js
  • Implemented features for users to add and play games

Core Features

AI-Generated Roadmaps

Integrated an AI to process exam data and generate personalized JSON roadmaps, guiding the learning path.

Gamified MCQ Engine

MCQ game with timed questions, hints, scoring (10+ for correct, 5+ with hint), and instant feedback.

User-Generated Content

Users can create and share their own learning games using a simple interface that accepts JSON data.
