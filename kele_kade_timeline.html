Kele-Kade Cycle Management

Internal Web Application for a Bicycle Repair Shop

Project Completed

Project Overview

Kele-Kade is a full-stack internal web application built to streamline operations for a local bicycle repair shop. The system provides real-time management of employees, products, and repair jobs, replacing manual tracking with an efficient, centralized digital solution. The platform was built using Next.js for the front-end and Appwrite as a Backend as a Service (BaaS) to handle the database, authentication, and real-time features.

Core Features

Real-time Employee Management

Track employee status, assign tasks, and monitor performance through a centralized dashboard.

Product & Inventory Control

Manage bicycle parts and product inventory with real-time updates on stock levels.

Repair Job Tracking

Create, update, and monitor the status of ongoing repair jobs from intake to completion.
